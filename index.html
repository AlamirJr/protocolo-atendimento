Operação

Atalhos e boas práticas

  • Use Salvar no site para persistir os dados — ficam visíveis para todos que acessarem este link.
  • Use Exportar Excel para baixar um arquivo .xlsx com todos os registros.
  • Use Importar Excel para carregar registros de um arquivo .xlsx salvo anteriormente.
  • Para consulta rápida, a tabela é ordenada por PSP (exibição).
  • A tabela mostra campos essenciais. Use Ver para detalhes completos.
  • Use Copiar pendências para gerar um resumo por PSP e colar no WhatsApp/e-mail.
  • Use o campo Filtrar acima da tabela para achar qualquer registro rápido.
  • Use Remover (linha vermelha na tabela) para excluir um atendimento específico.
